## Overview
This repository contains a case study of an end-to-end automated financial reporting solution built using Python, Power BI, and Pentaho.

Due to client confidentiality, this repository includes architecture, process flow, and documentation instead of client code.

## Key Features
- Automated data extraction from Microsoft Dynamics 365 using Python & Selenium
- ETL processing for bank transactions and trial balance
- Daily opening, inflow, outflow, and closing balance calculations
- Power BI dashboard with scheduled refresh
- Automated PDF export and email delivery
- Failure monitoring and alert notifications
